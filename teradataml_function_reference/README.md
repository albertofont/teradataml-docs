# teradataml Function Reference

This directory contains the API function reference for the Teradata Package for Python (`teradataml`). Files are organized with two-digit prefixes to preserve the original documentation ordering at https://docs.teradata.com/r/Enterprise_IntelliFlex_VMware/Teradata-Package-for-Python-Function-Reference-20.00/. Each Markdown file documents a chapter or a set of related APIs (classes, functions, utilities, analytic functions, etc.).

## Purpose

- Provide an exhaustive, local, and searchable API reference for `teradataml`.
- Serve as the primary source for syntax, parameter lists, return values, and short examples extracted from official Teradata Vantage documentation.

## How to use

- Open a numbered file, e.g. `03_teradataml_DataFrame_Object_and_Methods.md` to read about the `DataFrame` API.
- Search for function names across this directory to find signatures and examples.
